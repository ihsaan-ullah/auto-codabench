# How to participate

You should submit a .zip file containing a folder for each languages your model made a prediction for. Your model's prediction should be named test.system.cupt. The structure of your submission should be as follows:

```
submission.zip
├── DE
|   └── test.system.cupt
├── EL
|   └── test.system.cupt
├── EU
|   └── test.system.cupt
├── FR
|   └── test.system.cupt
├── GA
|   └── test.system.cupt
├── HE
|   └── test.system.cupt
└── HI
    └── test.system.cupt
```

This example showed a system that made predictions for 7 of the 14 languages. Making predictions for a subset of the languages causes your submission to have a zero score to unpredicted languages. All the languages are: 
```
DE,EL,EU,FR,GA,HE,HI,IT,PL,PT,RO,SV,TR,ZH
```
